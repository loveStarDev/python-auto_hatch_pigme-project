## 개요
○○지역아동센터, 코로나 상태로 인한 현장학습 또는 체험학습 불가로 인하여 제작한 프로젝트로, 해당 프로젝트를 통하여 실제 오리를 부화시키는 과정까지를 나타내었습니다.
